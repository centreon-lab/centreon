Host templates¶

Overview¶

Object name: HTPL

Refer to the HOST object

Note

You cannot use the APPLYTPL and SETINSTANCE actions on HTPL objects.

If you are looking for service templates association to host templates refer to ADDHOSTTEMPLATE/SETHOSTTEMPLATE command from STPL object.
